## Comece aqui

Prepare seu café e descubra o **Togglr**!

Nesta documentação você vai encontrar tudo o que precisa para conectar seus sistemas ao Togglr, nossa plataforma de gerenciamento de feature flags. Criado por engenheiros que também sofrem com deploys arriscados, o Togglr foi pensado para ser direto, prático e amigável para quem está no código todos os dias.

### O que é Togglr?

O Togglr é uma plataforma de feature flags que nasceu da nossa necessidade de controlar lançamentos sem dor de cabeça. Ao usar ferramentas existentes, percebemos alguns problemas comuns:

- Documentações enormes e pouco objetivas
- Integrações cheias de etapas desnecessárias
- Conceitos complicados para resolver problemas simples
- Sistemas que parecem exigir um treinamento formal só para ligar/desligar uma feature

Com o Togglr, a ideia é exatamente o oposto: ativar, desativar e experimentar funcionalidades sem fricção, sem burocracia e sem sustos em produção.
